My Projects:

  • Salieri: Pseudo-Homomorphic Computation Module

    The concept, protocol and definition of the new "pseudo-homomorphic" processor achitecture for trusted cloud computing, anti-tampering and computer science. Using the modern hardware anti-tampering technologies (which are pretty succesfull) and asymmetric crypto-algorithms, I developed the first open-source (GNU GPL v3) specialization of pseudo-homomorphic processor. And using the concept of "oracles", PHCM can also solve various computer science problems in practice ("Secure Multi-Party Computation" used as an example in this article). Article(RU)

  • LGCA: Lossless Geometrical Compression Algorithm

    Integer can be compressed by finding closest perfect power, then writing power and the absolute value of difference between them. Approximate compression rate (on random numbers) from 4 % to 21 %
