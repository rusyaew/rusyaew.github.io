Symmetrical Encryption: One Time Pad

You might have heard that ciphers are divided into symmetrical and non-symmetrical. In the picture, you can see symmetrical encryption. "Symmetrical" means that encryption and decryption use the same key.

1. One Time Pad

One Time Pad is unbreakable (mathematically proven) and requires a message and random key with exactly the same length. How it works?

Where mi , ki and ci is ith letter of message, key and ciphertext, respectively.

But how can we present the “exclusive or” operation in a more familiar form?

Python implementation:

import os
			
			
msg = 'foobar'
key = ''+os.urandom(6)
ciphertext = b''

for i in range(msg):
	m_i = ord(msg[i])
	k_i = ord(key[i])
	c_i = m_i ^ k_i
	ciphertext.append(chr(c_i))
			
print(ciphertext)
